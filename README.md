# Edouard Blondeau — Personal Website

Personal portfolio built using an AI-assisted product workflow.

The goal of this project was not only to build a website, but to explore how product thinking + structured prompting + AI collaboration can accelerate building real products.

👉 [edouard-blondeau.com](https://edouard-blondeau.com)

---

## Why this project exists

This repository documents a simple experiment:

Can a modern product workflow combined with AI drastically accelerate the creation of a real product?

Instead of starting directly with code, the project followed a product-first approach:

- define the product
- structure the prompts
- collaborate with AI during development
- iterate post-launch

The site itself is intentionally minimal. The real value lies in the process behind it.

---

## Product Philosophy

AI is most powerful when used inside a structured workflow, not as a simple code generator.

The goal of this project was to simulate a mini product lifecycle:

1. Product framing
2. Prompt engineering
3. AI-assisted development
4. Iterative improvements

This approach allows a single builder to move from idea to production extremely fast.

---

## Build Process

The website was built using a simple AI-assisted product workflow:
```bash PRD (Notion) → Prompt design (ChatGPT) → AI-assisted development (Claude + VS Code) → Launch (Vercel) → Post-launch improvements (Claude)```

1. Product framing (PRD)
Define the goal, audience, structure, design principles and technical constraints of the portfolio.

2. Prompt design (ChatGPT)
Translate the PRD into structured prompts covering UI, UX, content and technical guidelines for Claude.

3. AI-assisted development (Claude + VS Code)
Build and iterate on the site through a collaborative loop: generate components, refine UI, debug and adjust responsiveness.

4. Post-launch improvements (ChatGPT)
Optimize the production version with SEO, metadata, Open Graph preview, analytics, sitemap and cookie consent.

--- 

## What I learned building with AI

**Framing matters more than prompting**
Clear product framing significantly improves AI outputs.

**AI works best as a collaborator**
The most effective workflow is iterative:
human direction → AI implementation → refinement → iteration.

**AI excels at iteration**
It accelerates tasks such as UI refinements, SEO improvements, documentation and debugging.

**Product thinking remains essential**
AI speeds up execution but does not replace product strategy, prioritization or UX judgment.
