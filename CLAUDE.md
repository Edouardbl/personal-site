# Project: Edouard Personal Website

Goal:
Build a premium, minimal, editorial website for a senior product leader.
The site must feel calm, sharp, architectural, and extremely solid.

Design direction:
- Product rigor + editorial warmth
- No gradients
- No flashy animations
- Large whitespace
- Strong typography
- Subtle micro-interactions only

Tech constraints:
- Next.js App Router
- TypeScript
- Tailwind CSS
- No additional libraries unless strictly necessary
- Mobile-first
- Performance-first

Architecture:
- Clean components in /src/components
- Content stored as structured data (no CMS)
- Minimal abstraction
- Keep code readable and maintainable

Rules:
- Do not introduce unnecessary dependencies.
- Always explain which files will be modified before editing.
- Prefer simple solutions over complex patterns.
- Preserve the hero statement as-is.
